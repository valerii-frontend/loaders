# React loading components

## **Default props**

- `size`

  - **sm** - small size (50px)

  - **md** - medium size (100px) `// default`

  - **lg** - large size (150px)

- `speed` (animation speed) - **number** from **500** to **3000**

- `color`

  - **red** - HTML colors `// default white`

  - **#000** - HEX colors

  - **rgba(100,100,100,0.5)** - RGBA, RGB, HSL etc.
